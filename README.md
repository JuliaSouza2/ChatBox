Chatbot de Recursos Humanos desenvolvido no n8n durante a Imersão de Agentes de IA da Oracle Next Education (ONE) em parceria com a Alura.

## Tecnologias

- n8n
- Cohere
- MySQL
- AI Agent
- Vector Store

## Funcionalidades

- Responde dúvidas sobre RH
- Consulta funcionários no MySQL
- Utiliza base de conhecimento (Vector Store)
- Mantém contexto da conversa
